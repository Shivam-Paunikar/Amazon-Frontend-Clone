# Amazon Clone Frontend

This project is a frontend
clone of the Amazon website, built using HTML and CSS. It replicates the core interface of the e-commerce platform, including headers, navigation bars, product grids, and the footer section.

## Technologies Used 

- HTML5
- CSS
- FontAwesome (for icons)

## Key Features

-**Responsive Design:**
The
layout is structured to adapt to various screen sizes.
-**Navigation Panels:**
Includes the main search bar, cart, and category navigation.-**Shop Section:**Grid-based display boxes for different
product categories.
-**Detailed Footer:**Features
the classic Amazon footer
layout with numerous links.

## How to Run

1. Download all project files
(index.html', 'style.css', and the images' folder)into the same directory.
2.Open the index.html' file in your preferred web browser.
